# Bash Basics

Bash is a command-line shell commonly used in Linux, Git Bash, cloud environments, Docker containers, and CI/CD pipelines.

## Why Bash matters

Bash is important because many real engineering environments use Linux-based systems. Even if I work on Windows, I need to understand basic Bash commands so I can work with servers, automation scripts, GitHub Actions, and cloud tools.

## Basic command comparison

| Task                 | PowerShell                    | Bash                   |
| -------------------- | ----------------------------- | ---------------------- |
| Show current folder  | `pwd`                         | `pwd`                  |
| List files           | `ls`                          | `ls`                   |
| Move between folders | `cd folder`                   | `cd folder`            |
| Create folder        | `mkdir folder`                | `mkdir folder`         |
| Create file          | `New-Item file.txt`           | `touch file.txt`       |
| Read file            | `Get-Content file.txt`        | `cat file.txt`         |
| Copy file            | `Copy-Item file.txt copy.txt` | `cp file.txt copy.txt` |
| Move or rename file  | `Move-Item old.txt new.txt`   | `mv old.txt new.txt`   |
| Remove file          | `Remove-Item file.txt`        | `rm file.txt`          |
| Print text           | `Write-Output "Hello"`        | `echo "Hello"`         |

## Key difference

PowerShell is commonly used in Windows environments.

Bash is commonly used in Linux and Unix-like environments.

Both allow me to navigate folders, run commands, execute scripts, and automate tasks.
