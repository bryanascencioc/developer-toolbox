## 4. Concepts

### Bash

Bash is a command-line shell commonly used in Linux, Git Bash, cloud, Docker, and CI/CD environments.

### PowerShell vs Bash

PowerShell is common in Windows environments.

Bash is common in Linux-based environments.

Both are used to run commands, navigate folders, run scripts, and automate tasks.

### Why Bash matters for real environments

Bash matters because many servers, automation tools, containers, and CI/CD pipelines use Linux-based systems.
